# Chapter 9 Forms

* Create a new project with your :id: (i.e. b30000000 )

 ``` 
 $ ng new <ID> --style=scss && cd <ID>
 ```

* When asked answer as below:

```
? Would you like to add Angular routing? No
? Which stylesheet format would you like to use? SCSS
```
